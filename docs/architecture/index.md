# Architecture

Polygon is designed as a modular and structured ecosystem, with each component serving a specific purpose. Below is an
overview of the main components that make up Polygon:

**Library**

The utility library that provides common functions and helpers used across the ecosystem. The Polygon project will
utilize this.

**Command Line**

The CLI that commands the project and run various tasks.

**Server**

The repository that contains shared snippets and utilities for building backend services.
